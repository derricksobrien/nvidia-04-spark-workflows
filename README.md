# NVIDIA Spark Developer Workflows

> **Track:** NVIDIA HPC & Developer Upskilling (Track 2)
> **Course 4 of 8**

Accelerate Spark ETL and streaming workloads with RAPIDS cuDF and GPU-native execution.

---

## Modules

### Module 1: RAPIDS & cuDF Introduction

**Learning Objectives:**
- Replace pandas with cuDF operations
- Load and transform GPU DataFrames
- Measure cuDF vs. pandas speedup

### Module 2: GPU-Accelerated Spark on YARN

**Learning Objectives:**
- Configure Spark with RAPIDS plugin
- Set GPU resource scheduler parameters
- Monitor GPU utilization during jobs

### Module 3: ETL Pipelines at Scale

**Learning Objectives:**
- Build multi-stage GPU ETL pipelines
- Handle schema evolution in cuDF
- Optimize join operations on GPU

### Module 4: Spark Streaming with NVIDIA GPUs

**Learning Objectives:**
- Process Kafka streams with GPU acceleration
- Apply windowing functions on GPU
- Debug streaming lag and backpressure
